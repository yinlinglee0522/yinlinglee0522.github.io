The role I pick is designer, and the website I choose to perform the accessibility evaluation is Notable Alumni.

Suggestion 1

Increase the contrast between the color of the header text and its background.

Some of the header text, including global navigation and sub-navigation such as "HOME", "MAGAZINE ARCHIVE", and "NOTABLE ALUMNI" have low contrast between itself and its background. This website should (1) brighten the color of "HOME"; (2) darken the color of "MAGAZINE ARCHIVE" and "NOTABLE ALUMNI" to provide enough contrast.

Suggestion 2

Provide clearer signifiers regarding which element has the keyboard focus.

When users use the keyboard to navigate, the website displays a border around the user interface control which receives focus. However, this border is really thin so it is not visible enough. The website should show clearer focus states by making the borders thicker.

Suggestion 3

Rely on more characteristics as indicators for operating content.

The website only uses shapes — a circle and a plus sign — to indicate places for users to slide down the content hidden in each alumni section. The website should use a combination of more characteristics as indicators, such as shapes and captions saying "See more", to communicate the operations.

Suggestion 4

Include text for social media icons.

The social media icons in the footer are links containing no text, and this can cause confusion for screen reader users. The website should include text for these icons as additional screen reader-friendly experience to make this information more accessible.
